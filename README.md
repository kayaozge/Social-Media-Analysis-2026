# Social Media & User Behavior — Global 2026

End-to-end data analysis project examining social media usage
patterns, addiction levels, and mental health impact across
25,000 users and 17 platforms.

## Dashboard
[View Live Dashboard on Tableau Public](YOUR_TABLEAU_LINK_HERE)

## Key Findings
- **TikTok** leads on growth (+17% YoY), daily usage (58 min),
  and engagement rate (2.8%)
- **28.6%** of users report mostly negative mental health impact
- Positive correlation between screen time and addiction score
- TikTok users average the least sleep per night
